### ***git config***
- **git config --global user.name**
setup user name
- **git config --global user.email**
setup user's email
- **git config --list**
show files

### ***git init***
make empty git repository

### ***git status***
check working directory and staging area

### ***git add[file_name]***
changes to be comitted

### ***git add.***
every files changes to be comitted

### ***git rm --cached***
exclude file staging

### ***Ignoring a file***

### ***git log***

### ***git branch***
